Detailed Description

template<typename PathType = SingleVariate>
class Path< PathType >

Base class for a single path.

This Path class simply stores the information(e.g. time grid, stock price...) of a single path.

Template Parameters
PathTypeType of path(SingleVariate or MultiVariate).

How to use:
The path with antithetic path or control variable path will be derived from this class by using decorator pattern.
