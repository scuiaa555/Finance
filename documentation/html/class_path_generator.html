Detailed Description

template<typename RNG, typename PathType>
class PathGenerator< RNG, PathType >

Non-base class which uses a random number generator to generate a path for pricing.

The information of path that needs to be generated is stored in its data member model_(shared_ptr).

Template Parameters
RNGRandom number generator (e.g., Single<Normal<>>).
PathTypeType of path (SingleVariate or MultiVariate).

How to use:
It cannnot be derived from since it has no virtual member functions, but the RNG type and PathType need to be provided through template arguments and model_(information of model(e.g. LMM)) should be taken in through constructors.

§ next()

template<typename RNG , typename PathType >
const Path< PathType > & PathGenerator< RNG, PathType >::next ( )

Returns a newly-generated path.


Constness: Definitely it is not a const member function, since inside data member next_(shared_ptr) will be changed. Return reference type is const, since logically users cannot modify the path by using this reference.

Returns
Reference of a path.

Implementation involves several steps.

  1. It updates the initial value of path from model_;
  2. For each time t, model_ set up the arguments needed for the rng_(random number generator;
  3. rng_ generates a new sequence of random numbers;
  4. model_ evolves according to the random numbers.
