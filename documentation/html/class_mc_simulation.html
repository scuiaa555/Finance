Detailed Description

template<typename RNG>
class McSimulation< RNG >

Base class for all Monte Carlo pricing engines.

Any Monte Carlo pricing engine will be privately inherited(implement in terms of) from this class.
It will provide the mcModel class(its data member) with a specific path generator and a path pricer and control the calculation procedure of mcModel.

Template Parameters
RNGGeneric random number generator(e.g. SingleRandom<Normal<>>, MultiRandom<Normal<>,Poisson>,...)
