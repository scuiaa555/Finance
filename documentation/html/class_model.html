Detailed Description

template<typename Type_of_single_time>
class Model< Type_of_single_time >

Base class for all models(e.g.

Black-Scholes model, Merton's jump model, LMM, etc.). It is divided into two sub base classes.
One is Model1D simply for model which only contains 1-d underlying asset(e.g. Black-Scholes).
Another one is ModelND for multi-dimensional underlyings(e.g. LMM).
Any specific model class should derive from Model1D or ModelND due to its dimension.

Template Parameters
Type_of_single_timeValue type of one path(double or vector<double>).

Interface of evolve() when the required random number is only 1-d.

Parameters
t0Last time point, consistent with x0.
x0Value at time t0.
dtTime step.
dwRandom number required.
Returns
Value type of path at single time point.

Implementation: There are two versions of evolve() as interfaces, distinguished by type of random numbers dw. Each derived class should only override one of them, so they cannot be declared as pure virtual function. And they are supposed not to be called, so if they are called by mistake, they should throw "mismatch" error.
