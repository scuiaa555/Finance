Detailed Description

Base Instrument class.

This Instrument class is an abstract base class for any financial instruments, like options, forward, swap, swaption, etc. There is always a set of PricingEngine associated to each kind of instrument that can be used to price. Any inherited financial instrument classes should override the methods that are used to communicate the pricingEngines.
